Heuristics for Interface Analysis

Visibility of system status

Elite displays most of the vital system information on the main front window, such as ship shield status, power disribution, ship health and comms etc.. However there is still quite a lot of vital information that is rather hard to access quickly.

  • User control and freedom

    • Elites UI does not have much control and freedom, for example if the user presses the "eject all cargo" key, there is no confirmation.
